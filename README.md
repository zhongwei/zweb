# zweb

A lightweight desktop web app built with Rust, [wry](https://github.com/tauri-apps/wry) + [tao](https://github.com/tauri-apps/tao).

Frontend: Vue 3 · Tailwind CSS · FontAwesome 6

## Build

```bash
cargo build
```

Requires [WebView2 runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) (pre-installed on Windows 11).
